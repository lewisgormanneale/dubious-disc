FAQ

What Is This Tool For?

You can use the Pokémon Team Planner to plan out a team of Pokémon that you want to use in one of the mainline RPG series. While some players like to decide on team members as they go, this tool is designed to help those who prefer to plan in advance.

How Do I Use This Tool?

Select the game you're playing in the top left of the page, and the Pokémon in that game's Regional Pokédex will appear. To add a Pokémon to your planned team, simply click on it. You can filter the list of Pokémon by selecting from the options below the Team Analysis section. These options are designed to help you get type coverage, try and use new Pokémon, or ensure you have HM coverage.

How Does Prevent Type Overlap Work?

Selecting the Prevent Type Overlap button will automatically hide other Pokémon with the same types as any Pokémon that you've already added to your team.
